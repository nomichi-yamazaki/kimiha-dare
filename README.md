## 概要（ホラー小説）

『君は誰？』は、視点の反転によって意味が変わるホラー小説です。
一度目は恐怖として、二度目は全く別の物語として読める構造になっています。

違和感、不安、そして気づき。
この物語は、あなたの認識そのものに問いを投げかけます。

---

## この作品の特徴

* 二度読むことで意味が反転するストーリー構造
* 心理的恐怖を中心としたホラー表現
* 読者の解釈によって変わる結末の印象

## 読み方

* HTMLファイルをブラウザで開いて読むことを推奨します
* できれば一度で最後まで読んでください
* 読み終えた後、もう一度最初から読むことを強くおすすめします

## 作者より

この物語は、単なるホラー小説ではありません。

読むという行為そのもの、
そして「理解していると思っていること」に対する違和感を描いています。

もし読み終えたあとに、
ほんの少しでも世界の見え方が変わったなら——
それがこの作品の意味です。
